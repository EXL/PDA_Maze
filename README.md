PDA Maze
=============

PDA Maze is rewrite Bill Kendrick's and Robert Ernst maze game to modern Qt library.

Added:

1. New Compass Indicator;

2. Step Indicator;

3. Screen scale function with smoothing.

Fixed many graphical glitches.
